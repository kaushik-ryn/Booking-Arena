# 🚆 Booking System – All-in-One Travel Booking Platform

Booking System is a web application that allows users to search and book **Flights, Trains, and Buses** from a single platform.  
The platform provides a simple and responsive interface for checking routes, availability, and managing travel bookings efficiently.

This project demonstrates frontend development, UI design, and booking workflow simulation.

# 🚀 Features

## ✈️ Flight Booking
- Search flights between cities or airports
- Select departure and return dates
- Choose travellers and cabin class
- Multi-city route option
- Filter flights based on preferences

## 🚆 Train Booking
- Search trains between stations
- View popular trains
- Check departure and arrival timings
- Simple train search interface

## 🚌 Bus Booking
- Search buses between cities
- Check available routes and schedules
- View seat availability
- Booking flow simulation
